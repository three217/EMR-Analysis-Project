\### 论文信息

\*\*英文标题\*\*：Development and Validation of a Lightweight Large Language Model-Based Agent for Automatic Electronic Medical Record Structurization: A Single-Center Real-World Study  

\*\*中文标题\*\*：基于轻量级大型语言模型自动电子病历结构化代理的开发与验证：一项单中心真实世界研究  

\*\*论文状态\*\*：Under review（SCI期刊投稿中）

```bash

pip install -r requirements.txt



\###4. 下载基座模型

本项目基于 Qwen 2.5-3B-Instruct 开源模型开发，需提前下载模型文件：

下载地址：ModelScope Qwen2.5-3B-Instruct

下载完成后，记录模型文件夹的本地路径，用于后续系统配置。



\###5. 一键启动系统


python run.py (默认访问地址：http://localhost:8501)



\##核心功能（与论文系统架构完全对应）

1\.病历文书自动拆分：精准识别并拆分住院病历全类型文书，包括首次病程记录、医嘱单、检验报告、出院记录等。

2\.医学信息结构化抽取：针对拆分后的文书，完成临床关键字段的精准抽取，支持表格类检验报告的专项结构化处理。

3\.病历图片 OCR 识别：基于 PaddleOCR 优化的中文临床文本识别，支持图片格式病历的识别与结构化，适配检验单、医嘱单等多类型影像。

4\.医疗质控与逻辑校验：自动识别病历异常检验值、诊疗逻辑矛盾、文书缺失等问题，完成病历质量自动化校验。

5\.医疗隐私数据脱敏：自动对患者姓名、身份证号、病案号、联系方式等敏感信息完成脱敏处理，符合医疗数据安全规范。

6\.LoRA 微调训练：支持本地病历数据的一键式 LoRA 微调，优化模型在特定临床场景的适配能力。

7\.批量病历处理与导出：支持多文件批量处理，结果支持 JSON 格式导出与本地归档。



\##使用步骤:

1\.启动系统后，在侧边栏填写已下载的 Qwen2.5-3B-Instruct 模型本地路径，完成模型加载。

2\.输入病历数据：支持直接粘贴文本、上传 TXT 病历文件、上传病历影像图片三种输入方式。

3\.点击「开始病历结构化处理」，系统自动完成 OCR 识别、文书拆分、信息抽取、逻辑校验全流程处理。

4\.查看结构化结果，支持分文书卡片展示、表格展示、JSON 原始数据查看，可导出原始 / 脱敏结果，完成本地病历归档。
